npm・webpackの使い方

1. Node.jsのインストール
実行後、コマンドプロンプトで実行できるか確認。
node -v

実行できなければ再起動が必要かも

2. typescriptのインストール
npm install -g typescript

3. package.json, tsconfig.json, webpack.config.jsを準備する
このGitHubのRepositoryのものを使用する。必要なものは後で追加する。

ファイル配置: user-interface/src/main/resources

4. コマンドプロンプトでnpm installをする
cd user-interface/src/main/resources
npm install
